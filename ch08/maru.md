# 소프트웨어 설계란?

코드를 만들고, 테스트하고, 유지보수하기 쉬운 프로그래밍 방법을 선택하기 위해 미적 감각을 사용하는 것

# 계층형 설계란?

소프트웨어를 계층으로 구성하는 기술

각 계층에 있는 함수는 바로 아래 계층에 있는 함수를 이용해 정의

좋은 설계를 위한 감각은 어떻게 키울 수 있을까요?

계층화 설계 감각을 키우기 위한 입력 : 함수 본문, 계층 구조, 함수 시그니처

계층화 설계 감각을 키우기 위한 출력 : 조직화, 구현, 변경

# 계층형 설계 패턴

1. 직접 구현

2. 추상화 벽

3. 작은 인터페이스

4. 편리한 계층
